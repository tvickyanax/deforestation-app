# 🌍 Détecteur de Déforestation - Burkina Faso

## Description
Application de deep learning pour détecter la déforestation à partir d'images satellite.

## Performances
- Accuracy validation : **96.7%**
- Précision zone test : **70.2%**
- 42 588 patches d'entraînement

## Utilisation
1. Téléchargez une image satellite (JPG, PNG, TIF)
2. L'application analyse l'image
3. Résultat : zone boisée ou déforestée
